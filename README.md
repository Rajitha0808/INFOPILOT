# InfoPilot

![InfoPilot](header.jpg)

**InfoPilot** is an AI-powered conversational assistant that uses multiple LLM providers to generate comprehensive and intelligent responses. Instead of relying on a single model, InfoPilot creates an **"AI Council"** where multiple models collaborate, review, and rank each other's responses to deliver the best possible answer.

## How It Works

When you submit a query, InfoPilot processes it in three stages:

### 1. Stage 1: First Opinions
- The user query is sent to all LLMs individually.
- Each model generates its own response.
- Responses are displayed in a **tab view**, allowing users to compare them easily.

### 2. Stage 2: Review
- Each LLM reviews the responses generated by other models.
- Model identities are anonymized to ensure unbiased evaluation.
- Models rank responses based on **accuracy** and **insight**.

### 3. Stage 3: Final Response
- A designated **Chairman model** compiles all responses.
- It generates a final, refined answer based on collective insights.
